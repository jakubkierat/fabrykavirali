# Organizacja obrazów

W tej wersji wszystkie zdjęcia używane na stronie zostały zapisane lokalnie w folderze `assets/img/`.

Nie zmieniano wyglądu strony ani samych zdjęć — zostały tylko przeniesione z zewnętrznych adresów do lokalnych plików i podpięte w `index.html`.

Logo nadal znajduje się w `assets/fv-logo.png`, czyli również w folderze `assets`.
